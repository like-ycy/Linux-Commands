## 命令简介

ldd 命令打印程序和库的共享库依赖项。注意：ldd 不是一个可执行程序，而只是一个 Shell 脚本。

## 语法格式

```
ldd [OPTION]... FILE...
```

## 选项说明

```
--version  #打印指令版本号
-v  #打印所有相关信息
-u  #打印未使用的直接依赖
-d  #执行重定位和报告任何丢失的对象
-r  #执行数据对象和函数的重定位，并且报告任何丢失的对象和函数
--help  #显示帮助信息
```
