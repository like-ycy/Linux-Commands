## 命令简介

parted 命令用于创建，查看，删除和修改磁盘分区。它是一个磁盘分区和分区大小调整工具。这个命令算是对fdisk命令的一个补充，因为如果磁盘大小大于2TB就无法使用fdisk命令进行分区操作了。

## 语法格式

```
parted [options] [device [command [options...]...]]
```

## 选项说明

```
-a  #为新创建的分区设置对齐方式
-h  #显示帮助信息
-i  #交互式模式
-s  #脚本模式
-v  #显示版本号
-l  #列出所有块设备上的分区布局
-m  #显示机器可解析的输出
```
